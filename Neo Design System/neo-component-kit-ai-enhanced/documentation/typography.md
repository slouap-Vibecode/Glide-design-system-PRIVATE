The following typography classes should be used to style text:

Nota bene: you may encounter situations where the CSS environment imposes a stronger CSS rule wining over the following text styling rules. In such case, you may be able to overrule the conflicting rule by using the `.nck-style` css class as in: `<p class="nck-style header-1">Title</p>`

```js
<p class="amount-large">The wizard quickly jinxed the gnomes before they vaporized</p>
```
