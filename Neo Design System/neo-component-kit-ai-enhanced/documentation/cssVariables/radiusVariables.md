Available variables used for radius and border widths along with their initial values.

#### Border radius
- `--nck-radius-none`: 0
- `--nck-radius-s`: 0.25rem
- `--nck-radius-m`: 0.5rem
- `--nck-radius-l`: 1rem
- `--nck-radius-pill`: 999rem

#### Border width
- `--nck-border-width-none`: 0
- `--nck-border-width-s`: 0.0625rem (1px — default hairline border)
- `--nck-border-width-m`: 0.125rem (2px)
- `--nck-border-width-l`: 0.25rem (4px)
